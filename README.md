## 简介
一个基于Spring Boot和AOP的简单web后端框架。
主要是应用于接口通信的。也可以自己扩展。
_package demo.controller_ 已经写了一些简单明了的例子。
## AOP类功能
1. 接口加解密
2. 返回内容格式化
3. 鉴权
## 定时任务类
1. 初始化启动时，生成公私钥
2. 每秒更新时间戳（可能是多余的）

## 环境&依赖
+ jdk11+
+ redis
+ 建议docker。没有也可以。
+ 没有了，以后可能会增加

## 注意事项
+ 如果要对controller或者其他业务类方法使用aop包里的几个注解，强烈建议把所有相关方法的返回签名改成Object。毕竟原始类容易混水摸鱼嘿嘿。
